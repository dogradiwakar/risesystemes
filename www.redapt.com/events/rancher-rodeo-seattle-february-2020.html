Seattle Rancher Rodeo

February 12, 2020 | Seattle, WA

Join us for a free, hands-on interactive workshop focused on giving you the core skills needed to deploy highly-available, well-monitored and secure Kubernetes clusters. Hosted by Rancher Labs in Seattle.

Signup Now

Seattle Rancher Rodeo

  • Event: Seattle Rancher Rodeo
  • Date: Wednesday, Feb.12, 2020
  • Time: 9:30 a.m. â€“ 2:30 p.m.
  • Venue: AC Hotel Seattle Bellevue Downtown
  • Location: 208 106th Place NE BellevueWA 98004

Visit Event Website

Rancher Rodeos are free, in-depth workshops designed to give DevOps and IT teams the hands-on skills they need to deploy and manage Kubernetes everywhere. The content will be delivered by Rancher's technical experts and aim to educate anyone interested in learning how to use containers or Kubernetes.

 

During these hands-on workshops, one of our technical experts will provide an introduction to Rancher, Docker, and Kubernetes and then walk through the steps for deploying a Kubernetes cluster. The key topics that will be covered at the Rancher Rodeos include:

  • Docker and Kubernetes Concepts and Architecture
  • Installation and Configuration of Rancher Server
  • Kubernetes Cluster Deployment
  • Application Deployment and Access

 

What To Bring

  • Your laptop with a modern browser (Chrome, FF, Edge, IE8+). Training sessions are taught using an interactive, web-based tool called HobbyFarm. This tool lets you access compute resources through an in-browser terminal while following along with prepared learning scenarios.

 

AGENDA

10:00 a.m. - 10:45 a.m. Rancher Introduction & Marketplace Overview
10:45 a.m. - 11:15 a.m. Rodeo Introduction & Overview

  • Rodeo Objectives
  • Docker and Kubernetes Overview

11:15 a.m. - 11:45 a.m. Server Deployment

  • Machine Provisioning
  • Rancher Server Installation
  • Walkthrough of Rancher UI

11:45 a.m. - 12:15 p.m. Lunch Break
12:15 p.m. - 1:15 p.m. Kubernetes Deployment

  • Kubernetes Installation
  • Exploring the Clusters with Rancher and the CLI
  • Installing Applications from the Catalog

1:15 p.m. - 2:30 p.m. Q&A and Networking
