Virtual Workshop Series: Utilizing GPUs to Accelerate Machine Learning

Written by Redapt Marketing | May 6, 2020 4:58:00 PM

Gain the hands-on experience you need to bolster your AI and machine learning expertise.

Applying NVIDIA GPU Cloud (NGC) and Docker, you’ll master:

  • Downloading frameworks
  • Setting up notebooks
  • Programming and running models

You’ll come away with the ability to:

  • Run TensorFlow models in Docker on multiple GPUs
  • Use RAPIDS to build data pipelines and do feature engineering
  • Utilize TensorRT to deploy and optimize models for production
