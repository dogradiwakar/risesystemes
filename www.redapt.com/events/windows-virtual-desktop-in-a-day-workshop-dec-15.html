Windows Virtual Desktop (WVD) in a Day Virtual Workshop

December 15, 2020 |  9:00 a.m. - 4:00 p.m. Central Time (US & Canada)

 

Signup Now

Virtual Workshop Details

  • Event: Windows Virtual Desktop (WVD) in a Day Workshop
  • Date: Thursday, Dec. 15, 2020
  • Time: 9:00 a.m. - 4:00 p.m. Central Time (US & Canada)
  • Location: Virtual Workshop via Microsoft Teams

Visit Event Website

Microsoft Gold Partner Redapt is pleased to invite you to this upcoming Windows Virtual Desktop in a Day Workshop, and we encourage you to explore our Azure capabilities and reach out with any questions.

The Windows Virtual Desktop (WVD) Workshop provides you with both the knowledge and skills to accelerate designing and implementing a virtual desktop infrastructure on Azure. The workshop covers deployment options, architecture best practices, management plane options, user profiles and data, along with third party tools and licensing considerations. 

In the labs you will create your first WVD architecture, create Active Directory groups, assign users, create a master image and then connect to a virtual desktop. In the second lab you will dig into operationalizing your virtual desktop infrastructure with monitoring, scaling and image management.

We look forward to connecting with you!
