Utilizing GPUs to Accelerate Machine Learning with Redapt and NVIDIA

Written by Sarah Jones | Aug 5, 2019 6:10:22 PM

Please join Redapt and NVIDIA for a free one-day workshop to learn how to build machine learning models that are optimized and designed to run on NVIDIA GPUs.

When:  Thursday, August 22, 2019
9 a.m. to 4:00 p.m.

Location: Redapt Corporate Office
14051 NE 200th Street, Woodinville, WA 98072

Gain hands-on experience while you bolster your AI and machine learning expertise.

Applying NVIDIA GPU Cloud (NGC) and Docker, you’ll master:

  • Downloading frameworks
  • Setting up notebooks
  • Programming and running models


You’ll come away with the ability to: 

  • Run TensorFlow models in Docker on multiple GPUs
  • Use RAPIDS to build data pipelines and do feature engineering
  • Use TensorRT to deploy and optimize models for production
